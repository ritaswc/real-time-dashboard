## 旭日图 - 基于ECharts V4.2

旭日图（Sunburst）是ECharts 4.0新增的图表类型，由多层的环形图组成，在数据结构上，内圈是外圈的父节点。
因此，它既能像饼图一样表现局部和整体的占比，又能像矩形树图一样表现层级关系。

## 树图 - 基于ECharts V4.2

树图是一种流行的利用包含关系表达层次化数据的可视化方法。
